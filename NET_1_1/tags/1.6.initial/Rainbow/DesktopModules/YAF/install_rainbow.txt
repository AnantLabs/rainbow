ok. here you go:
1. YAF/Rainbow forum module is located in rainbowroot/DesktopModules/YAF
2. edit web.config and set <yafnet></yafnet> section. example:
        <yafnet>
        <connstr>data source=.\sqlexpress;Trusted_Connection=true;database=Rainbow;</connstr>
        <root>/Rainbow/DesktopModules/yaf/</root>
        <language>english.xml</language>
        <theme>standard.xml</theme>
        <uploaddir>/DesktopModules/Forum/upload/</uploaddir>
        <!--logtomail>email=;server=;user=;pass=;</logtomail-->
        </yafnet>
3. go to admin/modules definitions, go to add module, use install manually, give friendly name, set desktop source DesktopModules/yaf/rainbowmodule.ascx and mobile source for nothing.
4. add module to proper page
5. go to http://localhost/Rainbow/DesktopModules/yaf/install/Default.aspx. follow steps (if web.config is set right these steps lead to yaf forum under desktopmodules/yaf).
6. go to page where you added yaf forum moule and ther it is.

this rainbow yaf module is built on yaf 1.01 code. (http://sourceforge.net/projects/yafdotnet/)
